📚 Cyber Attack Glossary by Abid Mehmod

Malware

Malicious software designed to damage or gain unauthorized access to systems

Phishing

Fraudulent attempts to obtain sensitive information by disguising as trustworthy entities

DDoS

Distributed Denial of Service attacks that overwhelm systems with traffic

Botnet

Network of compromised computers controlled remotely by cybercriminals

Ransomware

Malware that encrypts files and demands payment for decryption

SQL Injection

Code injection technique that exploits vulnerabilities in database queries

Brute Force

Attack method that tries multiple password combinations to gain access

Zero-Day

Exploits that target previously unknown software vulnerabilities
