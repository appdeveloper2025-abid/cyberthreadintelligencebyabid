🚀 Getting Started

Our REST API provides access to real-time threat intelligence data. All endpoints require authentication via API key.

🔑 API Key

📡 Available Endpoints

GET /api/v1/threats

Retrieve current threat data with optional filtering

curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://api.threatintel.com/v1/threats
GET /api/v1/threats/{id}

Get detailed information about a specific threat

curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://api.threatintel.com/v1/threats/12345
POST /api/v1/ioc/lookup

Lookup indicators of compromise (IOCs)

curl -X POST \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     -d '{"indicator": "192.168.1.1", "type": "ip"}' \
     https://api.threatintel.com/v1/ioc/lookup
